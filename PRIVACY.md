# Privacy Policy / Politica de Privacidade

Last updated: 2026-03-02

## PT-BR

### 1) Escopo
Esta politica descreve como o OfflineNotes lida com dados no aplicativo Android.

### 2) Coleta de dados
O OfflineNotes **nao coleta dados pessoais**.

Nao utilizamos:
- analytics;
- telemetria;
- trackers;
- SDKs de publicidade;
- perfilamento de usuario.

### 3) Conexao de rede
O app foi projetado para funcionar offline no fluxo principal de notas.

Para o uso normal de anotacoes:
- nao ha envio automatico de conteudo para servidores;
- nao ha sincronizacao embutida com nuvem;
- nao ha upload de arquivos do usuario.

### 4) Armazenamento
As notas sao salvas na pasta escolhida pelo proprio usuario via Android SAF.

Formatos suportados:
- `.org`
- `.md`

Esses arquivos permanecem sob controle do usuario no dispositivo/armazenamento escolhido.

### 5) Titularidade dos dados
Todo e qualquer dado criado no app pertence ao usuario.

Em termos simples: **seus dados sao seus, apenas seus**.

### 6) Compartilhamento com terceiros
Nao vendemos, alugamos, compartilhamos ou transferimos dados de usuario para terceiros.

### 7) Seguranca
O app adota uma arquitetura local/offline para reduzir superficie de exposicao.
Ainda assim, a seguranca final do dispositivo e do sistema de arquivos depende do ambiente
do usuario (senha do aparelho, criptografia do dispositivo, backups externos etc.).

### 8) Alteracoes desta politica
Esta politica pode ser atualizada para refletir mudancas no aplicativo.
Quando isso ocorrer, a data de atualizacao sera alterada neste documento.

## EN

### 1) Scope
This policy explains how OfflineNotes handles data in the Android app.

### 2) Data collection
OfflineNotes **does not collect personal data**.

We do not use:
- analytics;
- telemetry;
- trackers;
- advertising SDKs;
- user profiling.

### 3) Network usage
The app is designed to work offline for the core notes workflow.

For normal note-taking usage:
- no automatic content upload to servers;
- no built-in cloud sync;
- no automatic user file transfer.

### 4) Storage
Notes are stored in a folder selected by the user via Android SAF.

Supported formats:
- `.org`
- `.md`

These files remain under user control on the selected device/storage.

### 5) Data ownership
All data created in the app belongs to the user.

In plain words: **your data is yours, and only yours**.

### 6) Third-party sharing
We do not sell, rent, share, or transfer user data to third parties.

### 7) Security
The app follows a local/offline architecture to reduce exposure surface.
Final security also depends on the user's environment (device lock, device encryption,
external backups, etc.).

### 8) Policy updates
This policy may be updated as the app evolves.
When that happens, the "Last updated" date in this file will be revised.

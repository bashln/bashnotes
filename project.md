# PROJECT

## Objetivo

OfflineNotes é um app Android de notas totalmente offline,
onde arquivos .md e .org são a única fonte de verdade.

## Filosofia

- Offline-first absoluto
- Privacidade acima de conveniência
- Simplicidade > features
- Arquivos como verdade, não banco

## Invariantes

- Sem rede, sync real ou telemetria
- Hamburger menu para navegaçao e acesso a menus
- SAF como único mecanismo de storage
- Nenhuma base de dados como fonte principal

## Prioridade Técnica

- Robustez de I/O
- Integridade do estado
- UX minimalista

## Fora de Escopo

- Backlinks
- Graph
- Login
- AI
- Sync embutido
